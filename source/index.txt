.. _atlascli:

==========================
What is the {+atlas-cli+}?
==========================

.. introduction::

   The {+atlas-cli+} is a command line interface built specifically for 
   |service|. It lets you interact with your |service| 
   {+database-deployments+} from the terminal with short, intuitive commands, so you can accomplish database management tasks in seconds.

   .. button:: Install the {+atlas-cli+}
      :uri: https://www.mongodb.com/docs/atlas/cli/stable/install-atlas-cli/

   Then, run ``atlas auth login``

.. image:: /images/hero.png
   :alt: Homepage hero image

.. kicker:: What You Can Do

Manage Atlas from the Command Line
----------------------------------

After you install and :ref:`connect from the {+atlas-cli+} <connect-atlas-cli>`, you can skip the Atlas UI and take action with a single command:

.. tabs::

   .. tab:: Sign Up for Atlas with ``atlas onboard``
      :tabid: onboard

      .. procedure::

         .. step:: Run ``atlas onboard``

            
         .. step:: Create a Profile

            Store your API keys, organization and project IDs, and other
            settings in a profile to easily access your MongoDB deployments.

            To learn more, see NEW PAGE.

      .. image:: /images/config.gif
         :alt: MongoDB CLI config command

   .. tab:: Create a Cluster with ``atlas quickstart``
      :tabid: create-cluster

      .. procedure::

         .. step:: Quick Start

            Create, configure and connect to an |service| cluster in one command.

            :doc:`To learn more, see Set Up Atlas with atlas quickstart <atlas-cli-quickstart>`

      .. image:: /images/quickstart.gif
         :alt: MongoDB CLI quickstart command

   .. tab:: Create Profiles with ``atlas config init``
      :tabid: create-profile

      .. procedure::

         .. step:: Create a Profile

            Store your API keys, organization and project IDs, and other
            settings in a profile to easily access your MongoDB deployments.

            :doc:`To learn more, see Save Connection Settings <atlas-cli-save-connection-settings>`

      .. image:: /images/quickstart.gif
         :alt: MongoDB CLI quickstart command
